========================================
  Asset Metadata Tool v1.0
  Standalone Windows Application
========================================

QUICK START:
  1. Double-click "AssetMetadataTool.exe" to launch
  2. That's it! No installation needed.

SYSTEM REQUIREMENTS:
  - Windows 10/11 (64-bit)
  - No Node.js or other software required

FEATURES:
  - Browse and manage asset files
  - View 3D models (OBJ, FBX, GLTF, GLB)
  - Preview images and videos
  - Edit metadata (batch or single file)
  - Export metadata to JSON
  - Resizable interface panels

TROUBLESHOOTING:
  - If Windows Defender blocks the app:
    Click "More info" → "Run anyway"
  
  - If the app won't start:
    Right-click AssetMetadataTool.exe → "Run as administrator"
  
  - Ensure all files in this folder stay together
    (Don't move just the .exe file)
